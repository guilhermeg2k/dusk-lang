# Roadmap

> This is a moving target and can change anytime

## Version: 0.6 - Takina

- Support for multiline function definition
- Anonymous structs
- Static struct methods & variables (consts)

## Version: 0.7 - Mizuhara

- Optionals
  - Optional fn args
- Function parameter default value
- Inference for function arguments (When it has default value)

## Version: 0.8 - Seras Victoria

- Error Handling

## Version 0.9 - Shinobu

- Fancy fors (ex: for in)
- Pipe operator
- Enum

## Version: 0.10 - Power

- Unions
- Pattern match

## Version 0.11 - Hiei

- Int/Float type
- Floor division
- Defer
- Ternary operator

## Version 0.12 - Kanade

- Multiline Comments
- Multiline strings
- Strings operations
- String interpolation

## Version 0.13 - Simon

- Upgrade to latest zig version
- Code Formatter

## Version 0.14 - Frieren

- Replace quickJS with Register-based VM

## Version 0.15 - Garou

- Closures (& fn as anything (argument, item, etc))

## Version 0.16 - Fujiwara

- Meta structs
- Generics

## Version 0.17 - Enjin

- Modules

## Version 0.18 - Yoruichi

- Collections: Lists, Sets, HashMaps, Tuples

## Version 0.19 - Shigeo

- Coroutines

## Backlog

- stdlib:
  - async io powered by libuv
- Fuzz testing
- Improve errors message
